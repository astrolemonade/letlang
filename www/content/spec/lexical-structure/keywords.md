# Keywords

Letlang divides keywords into 3 categories:

 - value keywords
 - operator keywords
 - syntactic keywords

# Value keywords

Those keywords are used to represent specific values:

 - `true`
 - `false`

# Operator keywords

Those keywords are used as part of Letlang operators:

 - `not`
 - `and`
 - `or`
 - `in`
 - `is`

# Syntactic keywords

Those keywords are use as part of the syntactic elements of the language, they
are classified in the following subcategories:

**Module declaration:**

  - `module`

**Import system:**

  - `from`
  - `import`
  - `as`

**Symbol declaration:**

  - `let`
  - `class`
  - `func`
  - `tailrec`
  - `effect`
  - `extern`

**Guards:**

  - `when`
  - `with`

**Control flow:**

  - `do`
  - `intercept`
  - `catch`
  - `perform`
  - `throw`
  - `spawn`
  - `receive`
  - `after`
  - `match`
  - `cond`
  - `else`

**Tail recursion:**

  - `recurse`
  - `final`
